Pertanyaan Umum (FAQ)

Bagaimana cara menambahkan tugas baru?

Mudah sekali! Ikuti langkah ini:

  1. Klik tombol ungu bertuliskan "+ Tambah Tugas" di pojok kanan atas halaman utama.
  2. Isi Judul Tugas (wajib).
  3. Pilih Kategori dan tentukan Deadline (wajib).
  4. Tambahkan catatan jika perlu, lalu klik tombol Simpan.

Tips: Prioritas tugas akan ditentukan otomatis berdasarkan seberapa dekat deadline-nya.

Apakah data saya aman?

Sangat aman. Tasking menggunakan LocalStorage browser Anda. Artinya, data tugas tersimpan langsung di perangkat Anda (laptop/HP) dan tidak pernah dikirim ke server kami.

Bagaimana cara menghapus kategori?

Buka menu "Kategori" (tombol putih di sebelah tombol Tambah Tugas). Di sana Anda akan melihat daftar kategori dengan ikon silang (x) kecil. Klik ikon tersebut untuk menghapus kategori.

Apa keuntungan fitur PRO?

Dengan hanya Rp 10.000/bulan, Anda bisa membuat kategori tanpa batas (Unlimited) dan mengubah warna tema aplikasi (Oranye, Pink, Teal) sesuai selera visual Anda.

Kenapa Google Calendar tidak muncul langsung?

Demi keamanan dan privasi data Anda, Google tidak mengizinkan akses langsung dari website tanpa login server. Kami menyediakan tombol "Buka GCal" untuk akses cepat ke kalender asli Anda.
